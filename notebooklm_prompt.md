# NotebookLM 抽出プロンプト集

NotebookLMのチャット欄に「★コピー開始★」から「★コピー終了★」の間を貼り付けてください。
出力結果は画面右上の **[↓IMPORT]** ボタンでそのままインポートできます。

**手順**: Prompt 1 実行 → コピー → IMPORT → Prompt 2 実行 → コピー → IMPORT
（パネル単位でマージされるため、2回インポートして1週分が完成します）

---

## ▼ ダッシュボード構成（参考）

| タブ | パネル | データの取得元 | 担当プロンプト |
|---|---|---|---|
| 1 TOPメッセージ | 石橋亮太 / 永田洋幸 / 牧草光一 / 内山智博 | 全配列を source名 でフィルタして表示 | Prompt 1 |
| 2 実務 | リゾート / アパレル / 秘書室 / AI関連 | resort・apparel・secretary + AI系キーワードで自動抽出 | Prompt 2 |
| 3 トピック | 市場トレンド / 数値情報 / 異常・懸念 | trends・numbers・anomalies | Prompt 1 |
| 4 プライベート | プライベート | private | Prompt 2 |

- Tab1は source の表記が表示に直結。4名の氏名は統一した表記で記載すること。
- AI関連パネルはキーワード自動抽出のため、抽出項目の追加は不要。

## ▼ Prompt 1：幹部・トピック（Tab1 + Tab3）

★コピー開始★

【抽出件数の設定】← 毎回ここの数値だけ変更してください

■ Tab1 TOPメッセージ（各人物の全カテゴリ合計件数）
  石橋　亮太 … 20件以上　永田　洋幸 … 20件以上
  牧草　光一 … 20件以上　内山　智博 … 20件以上

■ Tab3 トピック
  trends … 20件以上　numbers … 20件以上　anomalies … 20件以上

■ teams … 全員分

※AI関連は自動抽出。AI・DX関連記述をimportant/random/trendsへ積極的に含めること。

---

【件数は必達条件です】
上記の設定件数は最低件数であり、必ず達成してください。
資料を最初から最後まで丁寧に精読し、設定件数を満たすまで抽出を継続してください。
出力途中で件数が足りなくなっても、出力を止めずに資料から追加抽出して補完してください。
出力完了後に各配列の件数を数えて確認し、不足があれば即座に追記してください。

---

資料から以下7項目を抽出してください。
・text 30〜100文字・事実のみ・source=名前のみ（ファイル名不可）
・teams.text は「一言タグライン」形式のため文字数制限の対象外
・資料構造：「年週 グルーピング 名前」＋本文の繰り返し
・Tab1の4名は全カテゴリ合計が設定件数以上になるよう各配列に分散して抽出すること
・Tab1の4名のsourceは氏名のみ・表記揺れなしで統一すること

0.report: owner（氏名+役職）・period（対象期間）・title="経営ダッシュボード"
1.important: 経営最優先の事実・成果・決定。数値を積極的に含む。
2.anomalies: 未達・リスク・課題・異常。
3.random: 仕事関連の補足・雑感（私的な話はprivateへ）。
4.trends: 業界・競合・市場の外部動向。
5.teams: 人物の姿勢・動向。形式「テキスト」「source=氏名/役職」全員分。
6.numbers: 数値（売上・達成率・前年比等）を含む情報。

出力（○をすべて実データに置き換え・プレースホルダー削除）：
週キーは "YYYY-WNN" 形式（例："2026-W09"）で資料の対象期間から判断すること。

window.DASHBOARD_DATA = window.DASHBOARD_DATA || {};
window.DASHBOARD_DATA["YYYY-WNN"] = {
  "report":    {"owner":"○","title":"経営ダッシュボード","period":"○"},
  "important": [{"text":"○","source":"○"}],
  "anomalies": [{"text":"○","source":"○"}],
  "random":    [{"text":"○","source":"○"}],
  "trends":    [{"text":"○","source":"○"}],
  "teams":     [{"text":"○","source":"○"}],
  "numbers":   [{"text":"○","source":"○"}]
};

★コピー終了★

## ▼ 確認用プロンプト（文字数チェック）

★コピー開始★

各textについて以下の基準でチェックして再出力してください。
・100文字を超えているものは100文字以内に短縮すること
・30文字未満のものは意味を補完して30文字以上に拡張すること
・teams.text は対象外（タグライン形式のため）
・意味を損なわないこと・件数は変えないこと
